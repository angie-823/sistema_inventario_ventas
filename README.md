Este es un sistema informático profesional diseñado para gestionar el control de compras, ventas e inventario de un negocio.
El proyecto se desarrollará durante todo el tercer año de bachillerato para aprender a manejar bases de datos y flujos de trabajo profesionales.
Utilizaremos Git como nuestra "máquina del tiempo" para mantener un historial seguro de todos los cambios y mejoras del código.
